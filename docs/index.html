Quick Start

Up and running in 4 commands.

# 1. Clone
git clone https://github.com/choloville/mins-bot.git
cd mins-bot

# 2. Add your API key (minimum requirement)
echo "spring.ai.openai.api-key=YOUR_KEY" > application-secrets.properties

# 3. Build
mvn clean package -DskipTests

# 4. Run
mvn spring-boot:run

The floating orb appears on your desktop. Double-click to expand.
Access from any device on your network at http://your-pc-ip:8765/

Architecture

Spring Boot backend + JavaFX floating window + vanilla JS frontend.

☕

Java Backend

Spring Boot 3.5 serves the API and manages all AI orchestration, tool execution, memory, and integrations.

🪟

JavaFX Window

Transparent always-on-top window with WebView. No title bar. The orb is 64×64px, draggable anywhere on screen.

🎨

Vanilla JS Frontend

Pure HTML/CSS/JS served as static resources. No React or build step. Tabs, voice UI, streaming chat.

🔀

Dynamic Tool Routing

GPT-4o-mini classifies each message and selects the right subset of tools. Respects the 128-tool API limit.

🗂️

File-based Storage

No database. Persistent data lives in ~/mins_bot_data/ — plain text and JSON files.

🔌

Plugin System

Drop a new package under com.minsbot.skills.* and Spring auto-discovers it. No registration needed.
