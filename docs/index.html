Capabilities

Everything in one orb

Double-click to expand. Type, speak, or let it act on its own while you work.

👁️ Vision & Screen Awareness

Mins Bot captures your screen before every reply using GPT-5.4 vision. It knows exactly what's on your display — no hiding the window, no flickering. It just sees.

  • Live screen capture before every AI response
  • Detects dialogs, forms, errors, and notifications automatically
  • Bot window position passed as hint — vision ignores its own UI
  • Gemini 2.5 Flash as fallback, OCR as last resort
  • Webcam capture and analysis on demand
  • Screen memory — search what you were looking at 20 minutes ago
⚡ Proactive Action Mode

Turn on the lightning bolt and Mins Bot becomes fully autonomous. It watches your screen every 15 seconds, checks pending tasks every 30 seconds, and acts — all on its own.

  • Fills forms, dismisses dialogs, clicks buttons without you asking
  • Executes standing directives ("always copy links to clipboard")
  • 60-second cooldown between autonomous actions for safety
  • Speaks its actions aloud via TTS so you always know what it's doing
  • Jarvis Watch Mode — real-time commentary on your screen
  • Auto-pilot suggestions appear in chat as you work
🌐 Browser Control

Two browser engines, zero friction. Chrome DevTools Protocol gives the bot full control of your real Chrome session. Playwright handles headless automation in the background.

  • Navigate, click, scroll, type — all in real Chrome via CDP
  • JavaScript injection with browserExecuteJs — 10ms form fills
  • Extract text, links, images, structured data from any page
  • Headless Playwright for scraping without touching your browser
  • Web monitoring — alert when a page changes
  • fastPaste via Robot API for 200ms clipboard-paste injection
🤖 Background Agents

Launch up to 24 parallel AI agents on independent missions. Each gets its own memory, model selection, and tool set. You keep chatting while agents work in the background.

  • Per-agent model selection — GPT-5.4, GPT-4o-mini, Claude, Gemini
  • Isolated tool set: web search, files, Playwright, code runner
  • Live progress bar, log stream, and plan view in the Agents tab
  • Download agent results as formatted Markdown files
  • Cancel or remove agents at any time
  • Each agent gets its own conversation ID and memory
🎙️ Voice & Audio

Four TTS providers, real-time audio streaming, and background audio listening. Mins Bot can hear, speak, and translate — with near-zero latency.

  • ElevenLabs multilingual TTS (11_multilingual_v2)
  • Fish Audio with 1.1× speed and loudness normalization
  • OpenAI TTS and Windows native voice as fallbacks
  • Gemini Live — real-time bilingual audio streaming with translation
  • Web Speech API for browser-native voice input
  • Background audio capture + Whisper transcription
💾 Memory & Learning

Mins Bot remembers what matters. Life events are stored as searchable JSON episodes. Habits are detected from patterns. It learns which suggestions you like and improves.

  • Auto-memory extraction — pulls life facts from every conversation
  • Episodic memory — events with tags, people, mood, importance score
  • Habit detection from time/day patterns, persisted to disk
  • Feedback loop — rates suggestions, adjusts future behavior
  • Named skill workflows — trigger phrases execute saved action chains
  • App usage tracker — knows which apps you use most and for how long

Setup

Up in 4 commands

Requires Java 17 and Maven. One API key is enough to get started.

1
Clone the repo
git clone https://github.com/ceasis/mins-bot.git
2
Add your API key
Create application-secrets.properties with your OpenAI key. Gemini, Claude, and others are optional.
3
Build
mvn clean package -DskipTests
4
Run
The floating orb appears on your desktop. Double-click to expand. Access from phone at http://your-pc:8765
Full Setup Guide → 
      

      
# Clone
git clone https://github.com/ceasis/mins-bot.git
cd mins-bot

# Minimum config
spring.ai.openai.api-key=sk-your-key-here

# Optional extras
gemini.api.key=your-gemini-key
ANTHROPIC_API_KEY=your-claude-key
app.elevenlabs.api-key=your-tts-key

# Build & run
mvn clean package -DskipTests
mvn spring-boot:run
