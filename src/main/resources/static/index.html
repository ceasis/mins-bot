Agents

Parallel agents with limited tools only: web search, files (incl. Office/PDF), HTTP page fetch, and headless Playwright (isolated browser per operation — not your visible Chrome). No screen control, CDP, or shell.

Integrations

Google services below use the same OAuth client as TelliChat (spring.security.oauth2.client.registration.google.*). The redirect URI banner applies only to Google sign-in. Other cards are a roadmap — same layout, Coming soon. Bots (Telegram, Viber, WhatsApp, …) stay in application.properties; LLM keys in Setup.

Setup

Values are stored in application-secrets.properties next to the app. Empty fields are left unchanged. Use Forget to remove a saved value. Restart the app if chat, voice, or search still use old keys.
