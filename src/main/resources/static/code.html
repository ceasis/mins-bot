Code Generator

Mins Bot · scaffold projects via Claude Code CLI or the Anthropic SDK
Primary shells out to claude -p "<task>" --dangerously-skip-permissions in the working dir. Special calls the Anthropic API directly and writes the returned file manifest. Local uses Ollama (offline) with a locally-installed coder model (default qwen2.5-coder:7b). All runs every generator into its own subfolder so you can compare outputs side-by-side.
